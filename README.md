# *Library-demo*

It is a project made in java spring boot with an API Rest architecture which allows to borrow and return books from a library. 

>
##     Implementations:
>*      swagger.
>*      spring security.
>*      DTO.
>*      MySql.
>*      roles.
>*      Maven.

### url to run swagger: 
>
> localhost:8080/api-template/swagger-ui.html
